# Oliver's Learning Tracker

A console-based C# application developed for the CS690 Final Project.

The application helps users track books, save learning articles, and log reading sessions.

## Features

- Add and manage books
- Save online learning articles
- Start and end reading sessions
- Track pages read and time spent reading

## How to Run

Clone the repository:

git clone https://github.com/KSP-802/CS690-FinalProject.git

Navigate to the project:

cd CS690-FinalProject/src/OliversLearningTracker

Run the application:

dotnet run
